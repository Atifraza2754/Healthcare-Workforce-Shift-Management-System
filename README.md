# Healthcare Workforce Shift Management System

A Laravel-based workforce and shift management platform designed for healthcare staffing operations. The system enables administrators to manage shifts, facilities, worker compliance, payments, and reports through a centralized admin dashboard.

---

## Features

### Authentication
- Secure admin login system
- Profile and password management
- Role and permission settings

### Shift Management
- Create and manage shifts
- Approve, cancel, or delete shifts
- Open shifts tracking
- Calendar-based scheduling
- Shift invitations via secure token links

### Facility Management
- Facility approval and rejection
- Pending facility verification
- Facility management dashboard

### Compliance Management
- Worker certification tracking
- Expiring compliance alerts
- Expired credential monitoring

### Timesheets
- Worker timesheet tracking
- Attendance monitoring

### Payments
- Stripe payment integration
- Worker payout management
- Commission tracking
- Secure webhook handling

### Reports & Analytics
- Workforce reports
- Operational insights
- System analytics

### Audit Logs
- System activity tracking
- Administrative action monitoring
